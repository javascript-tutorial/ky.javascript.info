# 'if..else' дегенди '?' дегенге кайрадан жазыңыз

Бир нече тернарлык операторлорун `'?'` колдонуп, `if..else` түзүлүшүн кайра жазыңыз.

Окууга ыңгайлуу болуу үчүн кодду бир нече сапка бөлүүгө сунушталат.

```js
let message;

if (login == 'Кызматчы') {
  message = 'Салам';
} else if (login == 'Мүдүр') {
  message = 'Саламатсызбы';
} else if (login == '') {
  message = 'Логин жок';
} else {
  message = '';
}
```
