Тескери тырмакча `${...}` ичиндеги туюнтманы сапка киргизүүгө мүмкүндүк берет.

```js run
let name = "Ilya";

// туюнтма - 1 саны
alert( `салам ${1}` ); // салам 1

// туюнтма - "name" сабы
alert( `салам ${"name"}` ); // салам name

// туюнтма - өзгөрмө, аны сапка киргизебиз
alert( `салам ${name}` ); // салам Ильяс
```
